ğŸ“Œ Project Overview

The Patient Mortality Forecasting Framework is a SQL-driven healthcare analytics project focused on identifying critical factors influencing in-hospital patient mortality. The goal of this project is to transform raw clinical data into actionable insights that support early risk detection, operational optimization, and improved patient outcomes.

ğŸ¯ Business Objective

Hospitals often struggle to proactively identify high-risk patients during admission and treatment. Delayed interventions, unmanaged comorbidities, and ICU inefficiencies can significantly increase mortality rates.

This project aims to leverage structured data analysis to uncover mortality drivers and provide data-backed insights for clinical decision-making.
